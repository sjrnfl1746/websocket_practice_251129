# 💬 웹소켓 기반 실시간 채팅

## 📌 프로젝트 개요
- 프로젝트명 : 웹소켓 기반 실시간 채팅
- 개발인원 : 1명
- 목적 : 실시간 채팅 기능을 구현하며 웹소켓, STOMP, React, Spirng Boot의 동작 흐름을 이해

---

## 🚀 주요 기능
- 채팅방 입장/퇴장
- 실시간 메시지 송수신
- DB에 메시지 기록

---

## 🛠 기술 스택
|구분|사용기술|
|---|---|
|Frontend|React, Tailwind CSS, Axios, React Router, STOMP.js, SockJS|
|Backend|Spring Boot, MariaDB, JPA, Spring WebSocket|

## 📖 프로젝트 회고
- WebSocket을 활용하여 입장/퇴장과 메시지 전송이 실시간으로 반영되는 채팅 기능 구현에 성공함
- React의 useEffect와 useRef를 활용하여 자동 스크롤 기능을 구현, 사용자 경험(UX) 개선
- 향후 JWT 인증을 연동하여 로그인 기반 실시간 채팅으로 확장 가능성 확인
